## 第一阶段：成就系统 + 等级经验（立即可用）
1. 创建 `useAchievements` Hook：定义 12 枚徽章规则（首次专注、连续 3 天、深夜战士等）
2. 创建 `useLevel` Hook：每分钟=1XP，升级公式 `Math.floor(Math.sqrt(XP)/10)`
3. 创建 `AchieveBadge` 组件：Canvas 绘制动效弹窗，解锁时自动播放升级音效+粒子动画
4. 集成到 Timer 组件：番茄钟结束时自动检测成就并弹窗

## 第二阶段：分享卡片生成（传播利器）
1. 创建 `ShareCard` 组件：使用 Canvas 生成 1080×1080 精美卡片
2. 卡片内容：动态背景+专注数据+随机金句+站点品牌
3. 支持一键下载和 Web Share API 分享
4. 集成到成就弹窗：解锁成就后自动生成卡片

## 第三阶段：统计面板 + 微交互（沉浸感拉满）
1. 创建 `StatsPanel` 组件：使用 Chart.js 展示今日/本周/历史趋势
2. 数据存储：IndexedDB 存储专注记录，支持更大容量
3. 微交互优化：
   - 机械键盘音效（Web Audio 合成，可开关）
   - 番茄完成时屏幕轻微震动（CSS shake 动画）
   - 节日彩蛋（圣诞节飘雪等）
4. 数据导出：支持 JSON/CSV 格式一键下载备份

所有功能纯前端实现，零后端依赖，使用 LocalStorage + IndexedDB + Canvas 完成数据存储和图像生成。
