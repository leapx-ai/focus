# 🍅 Focus - 番茄工作法专注应用

一个美观、功能丰富的番茄工作法专注应用，帮助你提升工作效率，保持专注。

## ✨ 功能特色

### 🍅 番茄工作法
- **科学计时**：25分钟专注 + 5分钟短休 + 15分钟长休
- **动态波浪进度**：液体般的波浪动画，直观展示时间流逝
- **完成音效**：温馨的提示音，不打扰你的工作

### 🎵 白噪音系统
- **6种环境音效**：雨声、篝火、咖啡厅、海浪、白噪音、舒缓音
- **本地音频文件**：无需网络，即点即播
- **独立音量控制**：精确调节每个音效的音量

### 📝 任务管理
- **智能任务清单**：添加、完成、删除任务
- **详细统计**：总任务、完成率、今日/本周完成数
- **进度可视化**：直观的进度条和数据统计

### 🏆 成就系统
- **等级系统**：完成番茄钟获得经验值，提升等级
- **成就解锁**：达成特定条件解锁成就徽章
- **分享卡片**：生成精美的专注完成卡片，分享到社交媒体

### 🌟 其他特色
- **多主题支持**：默认、森林、日落、海洋等多种配色主题
- **中英双语**：一键切换中英文界面
- **键盘音效**：打字时的机械键盘音效（可开关）
- **响应式设计**：完美适配桌面和移动设备

- 📈 平均专注时长

- 📊 完成率统计

## 🎯 使用建议

### 最佳实践
1. **设定明确目标**：开始前明确这25分钟要完成的任务
2. **消除干扰**：关闭通知，告知同事你在专注时间
3. **适当休息**：严格按照番茄钟的节奏休息
4. **记录感受**：完成任务后简单记录完成感受

### 进阶技巧
- 结合白噪音营造最佳专注环境
- 使用任务清单管理每日待办事项
- 定期查看统计数据，了解自己的专注模式
- 通过分享卡片与朋友互相激励

### 主题配置

### 成就配置
在 `src/hooks/useAchievements.js` 中可以定义新的成就条件
