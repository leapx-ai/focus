# 🍅 Focus - 番茄工作法专注应用

一个美观、功能丰富的番茄工作法专注应用，帮助你提升工作效率，保持专注。

## 🌐 在线预览

访问在线演示：[https://leapx-ai.github.io/focus](https://leapx-ai.github.io/focus)

## ✨ 核心功能

- **🍅 科学计时**：支持标准番茄钟（25分钟）、短休（5分钟）、长休（15分钟）及自定义时长。
- **🌊 沉浸体验**：动态波浪进度背景、气泡视差动画，直观展示时间流逝。
- **🎵 白噪音**：内置雨声、篝火、咖啡厅等6种环境音效，支持独立音量控制。
- **📝 任务管理**：轻量级任务清单，支持任务统计与进度可视化。
- **🏆 成就系统**：包含等级成长、成就徽章解锁及精美的分享卡片生成。
- **🎨 个性化**：多主题切换（默认、森林、日落、海洋）、中英双语支持。

## 🚀 快速开始

### 本地开发

```bash
# 克隆项目
git clone https://github.com/leapx-ai/focus.git
cd focus

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 📦 部署

本项目已配置 GitHub Actions，推送到 `main` 分支即可自动构建并部署到 GitHub Pages。

- **构建命令**: `npm run build`
- **输出目录**: `dist`
